# Erste Schritte

Nach dem Einladen des Bots kannst du direkt loslegen — hier erfährst du, wie du auf das Dashboard zugreifst und die wichtigsten Einstellungen vornimmst.

## Dashboard öffnen

1. Gehe zu **[dashboard.dunebot.de](https://dashboard.dunebot.de)**
2. Klicke auf **Mit Discord anmelden**
3. Autorisiere den Zugriff auf deinen Discord-Account

## Server auswählen

Nach dem Login siehst du eine Übersicht aller Server, auf denen DuneBot aktiv ist und auf denen du Berechtigungen hast. Klicke auf den Server, den du konfigurieren möchtest.

## Dashboard-Aufbau

Das Dashboard ist in mehrere Bereiche unterteilt:

### Sidebar (links)
- **Übersicht** — Server-Statistiken auf einen Blick
- **Einstellungen** — Grundeinstellungen (Prefix, Sprache, Theme)
- **Benutzer & Gruppen** — Wer darf was im Dashboard?
- **Plugins** — Plugins aktivieren und konfigurieren
- **Themes** — Aussehen des Dashboards anpassen

### Hauptbereich (rechts)
Hier werden die jeweiligen Einstellungsseiten angezeigt.

### Header (oben)
- Server-Name und -Icon
- Schnellzugriff auf Discord-Server
- Benutzerprofil und Abmelden

## Grundeinstellungen

### Sprache ändern
1. Gehe zu **Einstellungen** → **Allgemein**
2. Wähle deine bevorzugte Sprache aus dem Dropdown
3. Klicke auf **Speichern**

### Prefix ändern
Der Standard-Prefix für Textbefehle ist `!`. Du kannst ihn ändern unter:
1. **Einstellungen** → **Allgemein**
2. Neuen Prefix eintragen (z.B. `.` oder `?`)
3. **Speichern**

> **Tipp:** Slash-Commands (`/befehl`) funktionieren immer, unabhängig vom gesetzten Prefix.

### Slash-Commands aktivieren
Slash-Commands sind standardmäßig aktiviert. Du kannst sie unter **Einstellungen** → **Allgemein** an- oder abschalten.

## Plugins aktivieren

Standardmäßig ist nur das **Core-Plugin** aktiv. Um weitere Funktionen zu nutzen:

1. Gehe zu **Plugins** in der Sidebar
2. Durchsuche die verfügbaren Plugins
3. Klicke auf **Aktivieren** beim gewünschten Plugin
4. Das Plugin erscheint jetzt in der Sidebar mit eigenen Menüpunkten

Mehr dazu: [Plugins verwalten](../dashboard/plugins-verwalten.md)
