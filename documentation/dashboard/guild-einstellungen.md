# Guild-Einstellungen

Die Grundeinstellungen deines Servers findest du unter **Einstellungen** → **Allgemein** im Dashboard.

## Allgemeine Einstellungen

### Prefix

Der Prefix wird für Textbefehle verwendet (z.B. `!help`). Slash-Commands funktionieren immer ohne Prefix.

- **Standard:** `!`
- **Erlaubt:** Beliebiges Zeichen oder kurze Zeichenkette (z.B. `.`, `?`, `db!`)
- **Empfehlung:** Verwende ein Zeichen, das auf deinem Server keine Konflikte mit anderen Bots verursacht

### Sprache

Legt die Sprache für Bot-Antworten und Dashboard-Texte fest.

- **Standard:** `de-DE` (Deutsch)
- Weitere verfügbare Sprachen werden im Dropdown angezeigt

### Theme

Wähle ein Theme für das Dashboard-Erscheinungsbild. Weitere Anpassungen sind über den [Theme-Editor](themes.md) möglich.

### Slash-Commands

Aktiviert oder deaktiviert Slash-Commands (`/befehl`). Wenn deaktiviert, funktionieren nur noch Textbefehle mit dem eingestellten Prefix.

- **Standard:** Aktiviert
- **Empfehlung:** Lass Slash-Commands aktiviert — sie bieten Autocomplete und sind benutzerfreundlicher

## Einstellungen speichern

Klicke nach dem Ändern auf **Speichern**. Die Änderungen werden sofort wirksam.

> **Hinweis:** Prefix-Änderungen wirken sich nur auf Textbefehle aus. Slash-Commands werden immer über `/` aufgerufen.
