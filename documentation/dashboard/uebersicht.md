# Dashboard-Übersicht

Das DuneBot Dashboard ist deine zentrale Anlaufstelle, um deinen Discord-Server zu konfigurieren. Hier findest du einen Überblick über alle Bereiche.

## Zugang

Das Dashboard erreichst du über **[dashboard.dunebot.de](https://dashboard.dunebot.de)** mit deinem Discord-Account. Du siehst nur Server, auf denen DuneBot aktiv ist und auf denen du entsprechende Berechtigungen hast.

## Navigation

### Sidebar

Die Sidebar auf der linken Seite ist dein Hauptmenü. Sie ist in mehrere Bereiche unterteilt:

**Kern-Bereiche** (immer sichtbar):
- **Übersicht** — Server-Dashboard mit Statistiken und Schnellzugriff
- **Einstellungen** — Prefix, Sprache, Slash-Commands
- **Benutzer & Gruppen** — Berechtigungsverwaltung
- **Plugins** — Plugin-Marketplace

**Plugin-Bereiche** (nach Aktivierung):
Jedes aktivierte Plugin fügt eigene Menüpunkte in die Sidebar ein. Zum Beispiel:
- Moderation → Fälle, Einstellungen
- Tickets → Ticket-Übersicht, Kategorien
- Gameserver → Server-Liste, Addons

### Header

Der Header oben zeigt:
- Den aktuell ausgewählten Server
- Schnellzugriff auf das Bot-Profil
- Deinen Account mit Abmelde-Option

## Übersichtsseite

Nach dem Auswählen eines Servers landest du auf der Übersichtsseite. Hier siehst du:

- **Server-Statistiken** — Anzahl Mitglieder, Kanäle, Rollen
- **Bot-Status** — Online/Offline, Ping, Uptime
- **Aktivierte Plugins** — Welche Plugins sind aktiv?
- **Letzte Aktivitäten** — Moderationsaktionen, Tickets, etc.

## Einstellungen

Unter **Einstellungen** → **Allgemein** findest du die Grundkonfiguration:

| Einstellung | Beschreibung | Standard |
|-------------|-------------|---------|
| Prefix | Prefix für Textbefehle | `!` |
| Sprache | Bot- und Dashboard-Sprache | `de-DE` |
| Theme | Dashboard-Erscheinungsbild | `default` |
| Slash-Commands | Slash-Commands aktivieren/deaktivieren | Aktiviert |

## Benutzer & Gruppen

Hier verwaltest du, wer im Dashboard Zugriff auf welche Funktionen hat:

- **Benutzer-Tab** — Liste aller Dashboard-Benutzer mit ihren Gruppen
- **Gruppen-Tab** — Gruppen erstellen, bearbeiten, Berechtigungen zuweisen

Mehr dazu: [Berechtigungen](../getting-started/berechtigungen.md)
