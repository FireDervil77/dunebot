# Statistik

Das Statistik-Plugin erfasst Benutzeraktivitäten und bietet Ranglisten und Rangkarten.

## Funktionen

- XP und Statistiken pro Benutzer
- Generierte Rangkarten (Bilder)
- Server-Ranglisten
- Level-Up-Benachrichtigungen

## Bot-Commands

| Command | Beschreibung |
|---------|-------------|
| `/stats [@user]` | Statistiken eines Benutzers anzeigen |
| `/rank [@user]` | Generierte Rangkarte als Bild |
| `/ranks` | Server-Rangliste (Top-Benutzer) |
| `/levelup` | Level-Up-Benachrichtigungen konfigurieren |
| `/statistics` | Detaillierte Server-Statistiken |

## Rangkarte

Der `/rank`-Befehl generiert ein individuelles Bild mit:
- Benutzername und Avatar
- Aktuelles Level und XP
- Fortschrittsbalken zum nächsten Level
- Position in der Server-Rangliste

## Dashboard-Einstellungen

Nach Aktivierung im Dashboard verfügbar:

- **Statistik-Übersicht** — Grafische Auswertung der Server-Aktivität
- **Einstellungen** — XP-Raten, Level-Up-Kanal, Rangkartendesign
