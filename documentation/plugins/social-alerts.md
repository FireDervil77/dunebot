# Social Alerts

Das Social-Alerts-Plugin benachrichtigt deinen Server automatisch, wenn Content-Creator auf Plattformen wie Twitch oder YouTube aktiv werden.

## Funktionen

- Automatische Benachrichtigungen bei Twitch-Streams
- YouTube-Video-Alerts
- Konfigurierbare Nachrichtenkanäle und -texte
- Mehrere Creator pro Server möglich

## Status

> **Hinweis:** Dieses Plugin befindet sich noch in der Entwicklung. Weitere Features und Einstellungen werden in kommenden Updates hinzugefügt.
