# Leveling

Das Leveling-Plugin belohnt aktive Mitglieder mit Erfahrungspunkten und Leveln.

## Funktionen

- Automatische XP-Vergabe für Aktivität (Nachrichten, Voice, etc.)
- Level-System mit konfigurierbaren Schwellenwerten
- Level-Up-Benachrichtigungen
- Rangliste der aktivsten Mitglieder

## Status

> **Hinweis:** Dieses Plugin befindet sich noch in der Entwicklung. Weitere Features und Einstellungen werden in kommenden Updates hinzugefügt.
