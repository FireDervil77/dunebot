# Voice Server

Das Voice-Server-Plugin ermöglicht die Verwaltung von Sprach-Kanälen und Voice-Servern direkt aus dem Dashboard.

## Funktionen

- Temporäre Voice-Kanäle erstellen
- Voice-Channel-Konfiguration

## Status

> **Hinweis:** Dieses Plugin befindet sich noch in der Entwicklung. Weitere Features und Einstellungen werden in kommenden Updates hinzugefügt.
