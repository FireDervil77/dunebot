# Clan Manager

Das Clan-Manager-Plugin ermöglicht die Verwaltung von Clans oder Gilden innerhalb deines Discord-Servers.

## Funktionen

- Clans/Gilden erstellen und verwalten
- Mitgliederverwaltung mit Rollen
- Clan-spezifische Kanäle und Berechtigungen

## Status

> **Hinweis:** Dieses Plugin befindet sich noch in der Entwicklung. Weitere Features und Einstellungen werden in kommenden Updates hinzugefügt.
