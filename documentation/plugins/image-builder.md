# Image Builder

Das Image-Builder-Plugin bietet automatische Bilderzeugung für verschiedene Zwecke wie Willkommensbilder, Rangkarten und mehr.

## Funktionen

- Automatische Bilderzeugung
- Vorlagen-System
- Integration mit anderen Plugins (Greeting, Statistik)

## Status

> **Hinweis:** Dieses Plugin befindet sich noch in der Entwicklung. Weitere Features und Einstellungen werden in kommenden Updates hinzugefügt.
